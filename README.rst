Future power
============
Calculating estimate power consumption price with given predictable model.



Package management
---------------------
`Poetry <https://python-poetry.org/>`_

Requirements
------------
- python 3.8.5
- modify ``.env.sample`` to ``.env`` and update variables


Installation
------------
.. code-block:: bash

    $ poetry install

Run
---
.. code-block:: bash

    $ poetry run mom-data

Test
----
.. code-block:: bash

    $ poetry run poe test


Style
-----
.. code-block:: bash

    $ poetry run poe style


Lint
-----
.. code-block:: bash

    $ poetry run poe lint


Build
-----
.. code-block:: bash
    
    $ poetry build
